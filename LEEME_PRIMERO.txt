CUENTA CLARA V9.0 - CUENTAS RECURRENTES Y COMPARTIDAS

Carpeta lista para reemplazar tu proyecto local:
C:\Users\caure\Desktop\Paginas\Cuentas

Cambios V9.0:
- Nueva sección Hogar / Recurrentes en la barra lateral.
- Permite crear una carpeta recurrente desde la cuenta actual, por ejemplo Streaming.
- Permite crear el siguiente mes conectado con el anterior.
- Si una persona queda pendiente, la deuda se arrastra al mes siguiente como “Deuda anterior”.
- Nueva sección Compartidas en la barra lateral.
- Permite publicar la cuenta actual como compartida.
- Permite invitar usuarios registrados por correo o nick.
- Los invitados aceptados pueden abrir y editar la cuenta compartida.
- La cuenta compartida se sincroniza en Supabase desde shared_accounts.
- Se mantiene el diseño V8.9: Historial arriba, Pagos sticky y perfil por secciones.

IMPORTANTE SUPABASE:
Para usar cuentas compartidas debes ejecutar este archivo nuevo en Supabase:
1. supabase-shared-accounts.sql

Si aún no lo hiciste antes, también ejecuta:
1. supabase-app-state.sql
2. supabase-social.sql
3. supabase-shared-accounts.sql

Instalación rápida:
1. Cierra el servidor local si está abierto.
2. Reemplaza los archivos de tu carpeta Cuentas por los de este ZIP.
3. Abre index.html con Live Server o usa tu flujo habitual.
4. Recarga el navegador con Ctrl + F5 para limpiar caché.

Nota:
Las cuentas recurrentes funcionan en local/nube personal con app_states. Las cuentas compartidas requieren iniciar sesión y ejecutar supabase-shared-accounts.sql.
