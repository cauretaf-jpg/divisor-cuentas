Privacidad

Privacidad y manejo de datos.

Esta política resume qué información puede usar Cuenta Clara, para qué se usa y qué debe considerar el usuario al guardar cuentas compartidas.

Última actualización: 30 de mayo de 2026.

Datos que puedes guardar

Cuenta Clara puede guardar nombres de cuentas, participantes, gastos, productos, propinas, pagos, estados de deuda, cuentas recurrentes, amigos, solicitudes, preferencias de uso y respaldos generados por el usuario.

Modo invitado

En modo invitado, los datos se guardan en el navegador del dispositivo mediante almacenamiento local. Si borras datos del sitio, cambias de navegador o usas otro dispositivo, puedes perder esa información.

Cuenta de usuario y sincronización

Al iniciar sesión, algunas funciones pueden sincronizar datos para perfil, amigos, solicitudes, cuentas compartidas y recuperación entre dispositivos. Esa sincronización puede usar servicios externos de base de datos o autenticación.

Cookies, almacenamiento local y funcionamiento técnico

El sitio puede usar cookies, almacenamiento local, service workers y tecnologías similares para mantener sesión, recordar preferencias, guardar datos de invitado, mejorar carga y permitir funciones de app web progresiva.

Publicidad de Google AdSense

Cuenta Clara puede mostrar anuncios mediante Google AdSense en páginas públicas del sitio. Google y sus socios pueden usar cookies o identificadores similares para publicar anuncios, limitar frecuencia, medir rendimiento y, cuando corresponda, personalizar publicidad según la configuración del usuario.

Control sobre anuncios personalizados

Los usuarios pueden revisar o modificar la personalización de anuncios desde las opciones de anuncios de Google y desde la configuración de privacidad del navegador. También pueden bloquear o borrar cookies desde su navegador.

Lectura de boletas

La lectura de boletas puede requerir procesar imágenes para detectar productos y montos. El resultado puede ser incompleto o incorrecto, por lo que siempre debe revisarse antes de guardar.

WhatsApp, correo y servicios externos

Cuenta Clara puede preparar mensajes para WhatsApp o correo. El envío final ocurre fuera de la app y requiere confirmación del usuario en la aplicación externa correspondiente.

Respaldos

Los archivos de respaldo exportados pueden contener información sensible de cuentas, personas y montos. El usuario debe guardarlos con cuidado y compartirlos solo si corresponde.

Seguridad y responsabilidad del usuario

Recomendamos no ingresar información bancaria, contraseñas, documentos de identidad ni datos innecesarios para dividir una cuenta. La app está diseñada para organizar gastos, no para almacenar información financiera sensible.

Contacto y solicitudes

Para consultas, reportes o solicitudes relacionadas con privacidad, usa la página de Contacto. Si escribes por correo, incluye el contexto necesario y evita enviar datos sensibles que no sean necesarios para resolver el caso.
