CUENTA CLARA V13.5 - LIMPIEZA TECNICA Y ESTABILIDAD

NUEVO EN V13.5

1. Limpieza de codigo repetido
- Se agrego el archivo shared-utils.js.
- Ahora script.js y profile.js comparten utilidades comunes para moneda, telefonos, fechas, iniciales, perfiles, amigos, HTML seguro y tema visual.
- Se mantuvieron las funciones publicas para no romper llamadas existentes.

2. Boton Aplicar plantilla corregido
- El boton Aplicar plantilla ahora ejecuta el cambio de plantilla seleccionado.
- Antes estaba definido en pantalla, pero no tenia accion conectada.

3. Limpieza de residuos antiguos
- Se eliminaron referencias a accountSettingsPanel y accountSettingsSummaryText, que ya no existen en el HTML actual.
- Se elimino la referencia antigua receiptButton, reemplazada por el flujo actual de boletas/OCR.

4. Mensajes mas profesionales
- Se reemplazo un mensaje tecnico que mencionaba SQL social.
- El usuario ahora ve un mensaje simple si no se pueden buscar usuarios registrados.

5. CSS depurado de forma segura
- Se eliminaron reglas de componentes antiguos sin uso actual.
- No se eliminaron overrides moviles o reglas historicas que podrian afectar la experiencia responsive.

6. Versionado y cache
- Version actualizada a V13.5.
- Service worker actualizado a cuenta-clara-v13.5.
- shared-utils.js agregado al cache.
- Sin SQL nuevo.
- Sin dependencias nuevas.

VALIDACIONES

- shared-utils.js: validado.
- script.js: validado.
- profile.js: validado.
- index.html: sin IDs duplicados.
- perfil.html: sin IDs duplicados.
- privacidad.html: sin IDs duplicados.
- styles.css: llaves balanceadas.
- Sin referencias activas a accountSettingsPanel, accountSettingsSummaryText, receiptButton ni SQL social.

PASOS PARA SUBIR

1. Descomprimir este ZIP.
2. Reemplazar los archivos en la carpeta local del proyecto.
3. En PowerShell, desde la carpeta del proyecto:
   git status
   git add .
   git commit -m "Publica Cuenta Clara v13.5"
   git push

4. Esperar el despliegue automatico de Vercel.
5. Probar desde celular.
