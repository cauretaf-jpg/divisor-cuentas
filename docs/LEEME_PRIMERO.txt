CUENTA CLARA V13.7 - RECURRENTES CON DEUDA ACUMULADA

NUEVO EN V13.7
- Cuentas recurrentes mas claras para hogar, streaming y gastos mensuales.
- Boton Crear siguiente mes con tratamiento de deuda pendiente.
- Opciones para deuda anterior:
  1) sumar al nuevo mes;
  2) dejar visible pero separada del total;
  3) marcar pagada en el mes anterior;
  4) ignorar esta vez.
- Historial mensual conectado por carpeta recurrente.
- Resumen de deuda acumulada por persona.
- Mensajes de WhatsApp con deuda anterior, mes actual y total acumulado.
- Plantillas recurrentes ampliadas para Streaming y Hogar.

COMO PROBAR
1. Abre index.html.
2. Crea una cuenta tipo Streaming u Hogar.
3. Agrega personas y gastos.
4. Marca al menos una persona como pendiente.
5. Activa la carpeta recurrente desde Hogar.
6. Usa Crear siguiente mes.
7. Prueba las opciones de deuda acumulada.

IMPORTANTE
- No requiere SQL nuevo.
- No agrega dependencias nuevas.
- Si el navegador conserva una version anterior, limpiar cache del sitio o abrir en ventana privada.

PUBLICACION SUGERIDA
1. Reemplazar archivos del proyecto.
2. Probar localmente.
3. Ejecutar:
   git add .
   git commit -m "Publica Cuenta Clara v13.7"
   git push
