CUENTA CLARA V13.4 - CENTRO FINANCIERO Y PAGOS PENDIENTES

NUEVO EN V13.4

1. Panel “Pendiente ahora” en Inicio
- Muestra acciones importantes sin entrar a cada sección.
- Resume cobros, pagos, vencimientos y revisiones críticas.
- Permite abrir rápidamente el Centro de pagos o revisar la cuenta actual.

2. Centro financiero en Pagos
- Nueva sección dentro de Pagos para revisar pendientes.
- Filtros disponibles: Cuenta actual, Todas las cuentas y Vinculadas a mi perfil.
- Cada pendiente permite Ver cuenta, Copiar mensaje, enviar WhatsApp y Marcar pagado.

3. Perfil financiero más accionable
- “A quién le debo” y “Quién me debe” ahora tienen botones directos.
- Desde el perfil se puede abrir la cuenta, copiar mensaje, enviar WhatsApp y marcar pagado.

4. Revisión previa antes de compartir
- Nueva tarjeta “Antes de compartir” en Resumen.
- Detecta personas faltantes, gastos faltantes, pagador principal pendiente, gastos sin consumidores, nombres repetidos, pagos pendientes y vencimientos del hogar.
- Ayuda a evitar enviar comprobantes incompletos o mal revisados.

5. Mantenimiento
- Versión actualizada a V13.4.
- Service worker actualizado a cuenta-clara-v13.4.
- Sin cambios de base de datos.
- Sin dependencias nuevas.

VALIDACIONES

- script.js: validado.
- profile.js: validado.
- index.html: sin IDs duplicados.
- perfil.html: sin IDs duplicados.
- No requiere ejecutar SQL nuevo.

PASOS PARA SUBIR

1. Descomprimir este ZIP.
2. Reemplazar los archivos en la carpeta local del proyecto.
3. En PowerShell, desde la carpeta del proyecto:
   git status
   git add .
   git commit -m "Publica Cuenta Clara v13.4"
   git push

4. Esperar el despliegue automático de Vercel.
5. Probar desde celular.
