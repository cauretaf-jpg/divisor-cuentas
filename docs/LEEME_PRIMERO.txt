CUENTA CLARA V11.4 - NAVEGACIÓN MÓVIL TIPO APP

NUEVO EN V11.4
- En celular, la app ahora usa una barra inferior fija con Personas, Gastos, Resumen y Pagos.
- Al entrar a cada sección móvil, se muestra como pantalla independiente.
- Se agregó encabezado móvil con botón Atrás.
- El menú de secciones lateral/superior se oculta en celular para que la pantalla quede más limpia.
- La barra móvil antigua de Total / Agregar / Compartir se ocultó para no competir con la nueva navegación.
- En escritorio se mantiene la navegación anterior.

CÓMO PROBAR
1. Sube esta versión a GitHub/Vercel.
2. Abre la app desde el celular.
3. Crea o abre una cuenta.
4. Usa la barra inferior: Personas -> Gastos -> Resumen -> Pagos.
5. En cada pantalla puedes volver con Atrás.

RECUERDA
Si el celular conserva la versión anterior, recarga con Ctrl+F5 en escritorio o borra datos del sitio/reinstala la PWA en el celular.
