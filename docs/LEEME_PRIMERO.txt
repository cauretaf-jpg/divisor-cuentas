CUENTA CLARA V13.6 - ONBOARDING Y PLANTILLAS INTELIGENTES

NUEVO EN V13.6

1. Asistente de plantillas en Inicio
- Ahora Inicio muestra una plantilla sugerida.
- Al tocar una plantilla, la app muestra una vista previa antes de crear la cuenta.
- El usuario ve descripcion, configuracion, beneficios y ejemplos.

2. Plantillas mas inteligentes
- Restaurante: propina activa, productos compartidos y comprobante WhatsApp.
- Supermercado: gastos del hogar o grupo, sin propina.
- Streaming: servicios mensuales y deuda pendiente arrastrable.
- Viaje: transporte, alojamiento, comida y gastos por etapa.
- Hogar: luz, agua, internet, gas, gastos comunes y cuentas recurrentes.
- Cuenta rapida: total unico dividido entre personas.
- Personalizada: configuracion libre.

3. Ayuda contextual en Gastos
- La app muestra la plantilla activa y el siguiente paso recomendado.
- Los ejemplos de gastos preparan el formulario, pero no agregan montos automaticamente.
- Esto reduce errores y evita cargar datos falsos.

4. Recomendacion automatica
- La app recomienda una plantilla segun la cuenta activa, uso reciente y contexto.
- Si el usuario selecciona otra plantilla, se respeta su seleccion.

5. Pulido adicional
- Se corrigio un texto duplicado en la tarjeta de cuenta actual.
- Version actualizada a V13.6.
- Service worker actualizado a cuenta-clara-v13.6.
- Sin SQL nuevo.
- Sin dependencias nuevas.

PASOS PARA SUBIR

1. Descomprimir este ZIP.
2. Reemplazar los archivos en la carpeta local del proyecto.
3. En PowerShell, desde la carpeta del proyecto:
   git status
   git add .
   git commit -m "Publica Cuenta Clara v13.6"
   git push

4. Esperar el despliegue automatico de Vercel.
5. Probar desde celular.
