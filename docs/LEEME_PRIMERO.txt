CUENTA CLARA V13.10 - RESPALDO Y SEGURIDAD MINIMA

NUEVO EN V13.10
- Se mantiene la experiencia visual de v13.9.
- Respaldo exportado con version, fecha, diagnostico y estado de la app.
- Importacion mas segura con vista previa antes de reemplazar datos.
- Antes de importar, la app crea una copia local previa para poder restaurar.
- Nueva seccion Respaldo y diagnostico dentro de Historial.
- Diagnostico simple: version, modo, cuentas, pendientes, sincronizacion y tamano local.
- Boton para copiar diagnostico.
- Boton para restaurar copia previa.
- Botones de respaldo visibles tambien en modo Simple.

COMO PROBAR
1. Abre index.html.
2. Entra a Historial.
3. Revisa la seccion Respaldo y diagnostico.
4. Presiona Exportar respaldo.
5. Importa un respaldo valido y revisa la vista previa antes de confirmar.
6. Despues de importar, prueba Restaurar copia previa.
7. Prueba importar un archivo JSON invalido: no deberia reemplazar tus datos.
8. Cambia entre modo Simple y Avanzado.
9. Revisa Inicio, Personas, Gastos, Resumen, Pagos y Perfil.

IMPORTANTE
- No requiere SQL nuevo.
- No agrega dependencias nuevas.
- Mantiene las funciones de v13.9.
- Recomendacion: exportar respaldo antes de subir una nueva version a GitHub/Vercel.
- Si el navegador conserva una version anterior, limpiar cache del sitio o abrir en ventana privada.

PUBLICACION SUGERIDA
1. Reemplazar archivos del proyecto.
2. Probar localmente.
3. Exportar respaldo antes de subir.
4. Ejecutar:
   git add .
   git commit -m "Publica Cuenta Clara v13.10"
   git push
